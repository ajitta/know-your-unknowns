# unknowns 플러그인 가치 계약

작성: 2026-07-11 · v0.3.0 사각지대 보고서 항목 "loop step 10 미적용" 대응.
loop 10단계(가치 리뷰: 지표 + 제거 조건)를 플러그인 자신에게 적용한 문서.

## 핵심 계약 (5줄)

1. **누구의 문제**: 작성자(및 설치자)가 강한 모델과 일할 때 지도(계획)–영토(현실) 격차를 보지 못하는 문제.
2. **가치 신호**: blindspot/loop 실행이 **사용자가 실제로 몰랐던 사실**을 드러내고, 그것이 **구현 결정을 바꿀 때**만 가치가 있다. 산출물 분량은 신호가 아니다.
3. **지표**: 실사용 10회 롤링 윈도우에서 "결정 변경 유발" 비율 ≥ 50% ([value-scorecard.md](value-scorecard.md)에 기록).
4. **강등 조건**: 특정 스킬이 10회 연속 결정 변경 0회면 해당 스킬을 README 추천 흐름에서 강등.
5. **제거 조건**: 두 릴리스 연속 스코어카드 기록 0건(= 실사용 없음)이면 플러그인 유지보수 중단을 선언한다.

수치(10회, 50%, 두 릴리스)는 측정을 시작하기 위한 임의 초기값이며 튜닝 대상.
